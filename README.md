# Kachin Visions Empire

A Progressive Web App for video streaming with secure password protection.

## Features
- Progressive Web App (PWA) - Installable on mobile/desktop
- Video streaming from multiple sources
- Secure password authentication
- Offline video downloading
- Admin management system
- Contact links management

## Quick Start

### Option 1: Use GitHub Pages (Recommended)
1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Set source to `/docs` folder
4. Your site will be available at: `https://[username].github.io/[repo-name]`

### Option 2: Deploy Manually
```bash
# Clone repository
git clone https://github.com/[username]/[repo-name].git

# Run deploy script
chmod +x deploy.sh
./deploy.sh

# Push to GitHub
git add .
git commit -m "Deploy application"
git push origin main
